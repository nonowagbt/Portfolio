# Portfolio Personnel

Un portfolio moderne et responsive créé avec HTML, CSS (Tailwind) et JavaScript.

## 🎨 Design

- **Couleurs principales** : Blanc comme couleur primaire, Rouge comme couleur secondaire
- **Style** : Moderne, épuré avec des animations fluides
- **Responsive** : Optimisé pour tous les écrans (mobile, tablette, desktop)

## ✨ Fonctionnalités

### Sections principales
- **Hero** : Section d'accueil avec effet de frappe et animations
- **Compétences** : Barres de progression animées pour les technologies
- **Projets** : Galerie de projets avec effets hover
- **Contact** : Formulaire de contact fonctionnel

### Animations
- Effet de frappe sur le titre principal
- Animations d'apparition au scroll
- Éléments flottants avec effet parallax
- Barres de progression animées
- Effets hover sur les cartes
- Barre de progression de scroll

### Responsive
- Navigation mobile avec menu hamburger
- Grilles adaptatives
- Optimisation pour tous les écrans

## 🚀 Utilisation

1. Ouvrez simplement le fichier `index.html` dans votre navigateur
2. Ou servez les fichiers via un serveur local pour une meilleure expérience

## 🛠 Technologies utilisées

- **HTML5** : Structure sémantique
- **Tailwind CSS** : Framework CSS pour le styling
- **JavaScript** : Interactions et animations
- **Font Awesome** : Icônes

## 📱 Compatibilité

- Chrome, Firefox, Safari, Edge
- Mobile et desktop
- Toutes les résolutions d'écran

## 🎯 Personnalisation

Vous pouvez facilement personnaliser :
- Les couleurs dans la configuration Tailwind
- Le contenu des sections
- Les animations dans `script.js`
- Les projets et compétences
